# Gaana-dl
Music and album downloader for Gaana.com

### How to use?
Make sure you have [python](https://www.python.org/ "python") installed in your system.

Clone this repo and navigate into its directory or follow the below commands (install [git](https://git-scm.com/) first):

```
git clone https://github.com/killua0102/Gaana-dl
cd Gaana-dl
pip install -r requirements.txt
```
```
python gaana.py https://gaana.com/album/bad-girls-like-you-english-version https://gaana.com/album/saiyaara-hindi-2025

```
Quality and download path can be changed from `config.py` file.
The files will be save in the ``Gaana-dl/Downloads/`` directory
